# Data Schema Spec

> Phase 1 deliverable. Defines what game data the site exposes and in what shape.

## Scope

- **Packs:** Turtle (free), Puppy, Star, Golden (paid) — static packs only
  - Weekly Pack excluded: it is a rotating subset of existing pack pets, not a source of unique pets
- **Localization:** English only for v1
- **Mechanics:** Yes — gold, XP, shop tier timing extracted from IL2CPP

---

## Data Types

### Pet

Core unit of the game. Pets have tiered stats and three leveled abilities.

```typescript
type Pet = {
  id: string;              // slug, e.g. "pet-ant" (mapped from numeric game ID)
  internalId: string;      // numeric string from game API, e.g. "88"
  name: string;            // display name, e.g. "Ant"
  tier: 1 | 2 | 3 | 4 | 5 | 6;
  baseAttack: number;
  baseHealth: number;
  packs: Pack[];           // which packs this pet appears in
  level1Ability: Ability;
  level2Ability: Ability;
  level3Ability: Ability;
  probabilities: Probability[];   // shop spawn rates per turn/pack
  image: ImageRef;
}
```

---

### Food

Items that can be bought in the shop or generated by pet abilities.

```typescript
type Food = {
  id: string;              // slug, e.g. "food-apple"
  internalId: string;      // numeric string from game API
  name: string;
  tier: 1 | 2 | 3 | 4 | 5 | 6;
  packs: Pack[];
  ability: Ability;        // single ability (no level variants)
  cost?: number;           // omit if standard cost (3 gold); present for exceptions (e.g. Sleeping Pill = 1, Milk = 0)
  image: ImageRef;
}
```

---

### Status

A persistent buff or debuff applied to a pet.

```typescript
type Status = {
  id: string;              // slug, e.g. "status-honey-bee"
  name: string;
  ability: Ability;        // describes the passive effect
  image: ImageRef;
}
```

---

### Ability

Polymorphic — `effect.kind` determines additional fields.

```typescript
type Ability = {
  description: string;     // human-readable text
  trigger: AbilityTrigger;
  triggeredBy: TriggerSource;
  effect: Effect;
}

type AbilityTrigger =
  | "AfterAttack" | "BeforeAttack" | "Buy" | "BuyAfterLoss"
  | "BuyFood" | "BuyTier1Animal" | "CastsAbility" | "EatsShopFood"
  | "EndOfTurn" | "EndOfTurnWith3PlusGold" | "EndOfTurnWith4OrLessAnimals"
  | "EndOfTurnWithLvl3Friend" | "Faint" | "Hurt" | "KnockOut"
  | "LevelUp" | "Sell" | "StartOfBattle" | "StartOfTurn" | "Summoned";

type TriggerSource = {
  kind: "Self" | "FriendAhead" | "FriendBehind" | "EachFriend" | "Player" | string;
}

// Effect is polymorphic on `kind`
type Effect =
  | { kind: "AllOf"; effects: Effect[] }
  | { kind: "OneOf"; effects: Effect[] }
  | { kind: "ModifyStats"; target: Target; attackAmount?: number; healthAmount?: number; untilEndOfBattle?: boolean }
  | { kind: "DealDamage"; target: Target; amount: number }
  | { kind: "SummonPet"; pet: string; withAttack?: number; withHealth?: number; level?: number; team?: string }
  | { kind: "SummonRandomPet"; tier?: number; level?: number }
  | { kind: "ApplyStatus"; status: string; to: Target }
  | { kind: "GainGold"; amount: number }
  | { kind: "GainExperience"; target: Target; amount: number }
  | { kind: "TransferStats"; from: Target; to: Target }
  | { kind: "TransferAbility"; from: Target; to: Target; level?: number }
  | { kind: "RepeatAbility"; target: Target; level?: number }
  | { kind: "FoodMultiplier"; multiplier: number }
  | { kind: "DiscountFood"; amount: number }
  | { kind: "RefillShops" }
  | { kind: "Evolve"; into: string }
  | { kind: "Swallow" }
  | { kind: "ReduceHealth"; target: Target; amount: number }
  | { kind: string; [key: string]: unknown }  // catch-all for future kinds
```

### Turn

Shop configuration per turn number.

```typescript
type Turn = {
  id: string;              // e.g. "turn-1"
  index: number;           // 1-based
  name: string;
  animalShopSlots: number;
  foodShopSlots: number;
  livesLost: number;
  tiersAvailable: number;
  levelUpTier: number;
}
```

---

### Mechanics

Game constants extracted from IL2CPP.

```typescript
type Mechanics = {
  goldPerTurn: number;              // 10
  xpForLevel2: number;              // 2
  xpForLevel3: number;              // 3 (cumulative: 5 total)
  maxAttack: number;                // 50 (can go to 100 per game message)
  maxHealth: number;                // 50
  shopTierUnlockTurns: number[];    // index 0 = tier 1 unlock turn, etc.
  // additional constants TBD from IL2CPP dump
}
```

---

### Pack

```typescript
type Pack =
  | "TurtlePack"      // free base game
  | "PuppyPack"       // paid
  | "StarPack"        // paid
  | "GoldenPack"      // paid
  | string;           // forward compat for new packs
```

> **Note:** Internal API pack names may differ. Known mappings so far: `StandardPack` → `TurtlePack`, `ExpansionPack1` → `PuppyPack` (to be confirmed from live API data in Dev Phase 1).

---

### Supporting types

```typescript
type Probability = {
  kind: "shop" | "levelup";
  turn: number;
  pack: Pack;
  probability: number;
}

type ImageRef = {
  source: string;  // path within extracted assets, or URL
}

type Target = string; // e.g. "Self", "RandomFriend", "FriendAhead", "EachEnemy", etc.
```

---

## Output Format

All data is stored as JSON files in `data/extracted/`:

```
data/extracted/
  pets.json         — map of id → Pet
  foods.json        — map of id → Food
  statuses.json     — map of id → Status
  turns.json        — map of id → Turn
  mechanics.json    — Mechanics constants
  changes.json      — raw Changes.json from latest bundle (patch diff)
  assets/
    sprites/        — PNG files named by slug (e.g. pet-ant.png)
```

---

## What We Are NOT Exposing (v1)

- Cosmetic hats (not gameplay-relevant)
- Localization strings in non-English languages
- Replay/match data
- Player account data
